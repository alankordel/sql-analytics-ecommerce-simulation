# SQL Analytics Project – E-commerce Simulation

---

## Objective  
This project simulates analytical queries and data modeling practices using a fictional e-commerce database.  
The goal is to demonstrate SQL proficiency, analytical thinking, and foundational data engineering concepts.  

**Objetivo**  
Este projeto simula consultas analíticas e práticas de modelagem de dados utilizando um banco de dados fictício de e-commerce.  
O objetivo é demonstrar proficiência em SQL e conceitos fundamentais de engenharia de dados.
